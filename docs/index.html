苦手フレーズの傾向

マークした文・単語から、聞き取りにくさの原因（連結・脱落・弱形など）を分析し、似た音の練習フレーズを生成します。

単語帳 0

練習中にタップした単語・イディオムが「その文での意味」つきで並びます。意味はGeminiが文脈に即して判定します。

Gemini API 無料枠でOK

キーは端末内（localStorage）にのみ保存され、採点・分析時のみGoogleへ送られます。個人利用向けの簡易構成です。取得は aistudio.google.com/apikey から。

音源モード

現在は v0.1 デモ。実際のポッドキャスト音声は、次段の GitHub Actions パイプライン（RSS→Whisperで文字起こし＋タイムスタンプ→Geminiで対訳）が出力するJSONを読み込む形で差し込みます。デモでは端末の音声合成で全機能を試せます。

このプロトタイプについて

サンプル1エピソード同梱。録音はMediaRecorderで取得しWAV化してGeminiへ送信。採点はキー未設定時はサンプル表示になります。復習データは端末内に保存されます。
